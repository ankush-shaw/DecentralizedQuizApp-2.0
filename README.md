# 🧠 Decentralized Quiz App (Soroban - Stellar)

## 📌 Project Description
The Decentralized Quiz App is a blockchain-based quiz platform built using Soroban smart contracts on the Stellar network. It ensures transparency, fairness, and tamper-proof quiz results by storing questions, answers, and scores on-chain.

---

## 🚀 What It Does
- Allows admins to add quiz questions with correct answers
- Enables users to attempt quizzes in a decentralized way
- Automatically validates answers using smart contracts
- Stores and tracks user scores securely on-chain

---

## ✨ Features
- 🔒 Decentralized & Tamper-proof
- ⚡ Fast and low-cost transactions (Stellar network)
- 📊 On-chain score tracking
- 👤 User-based scoring system
- 🧩 Simple and extendable contract logic

---

## 🛠 Tech Stack
- **Blockchain:** Stellar
- **Smart Contracts:** Soroban (Rust)
- **Language:** Rust

---

## 📦 Smart Contract Functions

| Function | Description |
|--------|-------------|
| `add_question` | Add a new quiz question |
| `get_question` | Fetch a question |
| `submit_answer` | Submit an answer and update score |
| `get_score` | Get user's score |
